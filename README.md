**Frontend:** Next.js 15, TypeScript, TailwindCSS  
**Backend:** FastAPI, Python, PostgreSQL  
**ML:** XGBoost, SHAP, LIME, Optuna  
**Deployment:** Netlify + Railway + Supabase
